# 🎉 Funny Birthday Wishes Website 🎉

A fun, interactive birthday website with animations and entertaining button interactions!

## Features

- **🎊 Confetti Button**: Creates colorful confetti particles that fall with physics
- **🎈 Balloons Button**: Generates floating balloons that you can pop by clicking
- **💃 Dance Button**: Makes the page elements dance with random movements
- **🎁 Surprise Button**: Shows random funny birthday messages
- **🌪️ Shake Button**: Shakes the entire page for a fun effect
- **🎵 Music Button**: Plays a "Happy Birthday" melody using Web Audio API

## Setup

1. Simply open `index.html` in a modern web browser
2. No build tools or dependencies required!
3. Works best in Chrome, Firefox, Safari, or Edge

## How to Use

- Click any button to trigger its fun animation
- Buttons have cooldown timers to prevent spam clicking
- Balloons can be popped by clicking on them
- Enjoy the festive animations and funny messages!

## Technologies Used

- Pure HTML5
- CSS3 with keyframe animations
- Vanilla JavaScript (ES6+)
- Canvas API for particle effects
- Web Audio API for sound

## Browser Compatibility

Works in all modern browsers that support:
- CSS3 animations
- Canvas API
- Web Audio API (for music button)
